Perguntas Frequentes

O que é um componente "accordion"?

Um accordion (ou sanfona) é um elemento de interface que permite mostrar e esconder seções de conteúdo. Ele é útil para organizar grandes volumes de informação em um espaço compacto, como em uma página de FAQ.

A principal vantagem é que o usuário pode focar no conteúdo que lhe interessa, sem se distrair com o resto.

Preciso de JavaScript para criar um?

Não mais! Antigamente, JavaScript era essencial. Hoje, o HTML5 oferece as tags <details> e <summary>, que criam um accordion funcional nativamente, sem nenhuma linha de script.

Isso torna o componente mais leve, rápido e acessível por padrão.
